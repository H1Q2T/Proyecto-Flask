Comprensiones en Python

Las comprensiones son una forma concisa de crear listas, diccionarios o conjuntos en una sola línea. Son útiles, legibles y muy potentes.

Listas por comprensión

En vez de usar un bucle tradicional, podemos crear una lista directamente así:


cuadrados = [x**2 for x in range(1, 6)]
print(cuadrados)

Resultado:


[1, 4, 9, 16, 25]

Filtrar elementos en una comprensión

También puedes añadir condiciones dentro de la comprensión:


pares = [x for x in range(10) if x % 2 == 0]
print(pares)

Resultado:


[0, 2, 4, 6, 8]

Diccionarios por comprensión

Igual que con listas, también puedes generar diccionarios directamente:


cuadrados_dict = {x: x**2 for x in range(1, 6)}
print(cuadrados_dict)

Resultado:


{1: 1, 2: 4, 3: 9, 4: 16, 5: 25}

Conjuntos por comprensión

También puedes generar un set directamente desde una cadena o lista:


letras = {letra for letra in "banana"}
print(letras)

Resultado:


{'b', 'a', 'n'}

Las comprensiones son una forma elegante de transformar o filtrar datos en una sola línea de código.
