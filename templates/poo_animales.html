Programación Orientada a Objetos: Animales

Este ejemplo real muestra cómo aplicar clases y herencia en Python para representar distintos tipos de animales según su alimentación.

Clase base: Animal


class Animal:
    def __init__(self, nombre):
        self.nombre = nombre

    def alimentarse(self):
        print("Este animal se alimenta.")

Subclase: Herbivoro


from animal import Animal

class Herbivoro(Animal):
    def alimentarse(self):
        print(f"{self.nombre} se alimenta de plantas.")

Subclase: Carnivoro


from animal import Animal

class Carnivoro(Animal):
    def alimentarse(self):
        print(f"{self.nombre} se alimenta de carne.")

Subclase: Omnivoro


from animal import Animal

class Omnivoro(Animal):
    def alimentarse(self):
        print(f"{self.nombre} se alimenta de todo.")

Uso de las clases


from carnivoro import Carnivoro
from herbivoro import Herbivoro
from omnivoro import Omnivoro

leon = Carnivoro("León")
vaca = Herbivoro("Vaca")
oso = Omnivoro("Oso")

leon.alimentarse()
vaca.alimentarse()
oso.alimentarse()

Resultado esperado:


León se alimenta de carne.
Vaca se alimenta de plantas.
Oso se alimenta de todo.

Este ejemplo demuestra cómo usar clases y herencia para organizar el código y reutilizar comportamientos de forma lógica y clara.
