Condicionales en Python

Las condicionales permiten ejecutar diferentes bloques de código según ciertas condiciones. En Python se utilizan las sentencias if, elif y else.

Ejemplo básico:


edad = int(input("Introduce tu edad: "))

if edad >= 18:
    print("Eres mayor de edad.")
elif edad >= 13:
    print("Eres un adolescente.")
else:
    print("Eres un niño.")

Explicación:

  • Si la edad introducida es mayor o igual a 18, se imprime que es mayor de edad.
  • Si no, pero es mayor o igual a 13, se considera adolescente.
  • Si no cumple ninguna de las anteriores, es un niño.

Resultado esperado (ejemplo):


Introduce tu edad: 15
Eres un adolescente.

Este tipo de estructuras es muy útil para controlar el flujo del programa dependiendo de decisiones del usuario o del estado de las variables.
