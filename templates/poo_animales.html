Programación Orientada a Objetos: Animales

La Programación Orientada a Objetos (POO) nos permite modelar el mundo real mediante clases y objetos. En este ejemplo, representamos animales que se alimentan de diferentes formas.

Clase base: Animal


class Animal:
    def __init__(self, nombre):
        self.nombre = nombre

    def alimentarse(self):
        print("Este animal se alimenta.")

Subclase: Herbivoro


from animal import Animal

class Herbivoro(Animal):
    def alimentarse(self):
        print(f"{self.nombre} se alimenta de plantas.")

Subclase: Carnivoro


from animal import Animal

class Carnivoro(Animal):
    def alimentarse(self):
        print(f"{self.nombre} se alimenta de carne.")

Subclase: Omnivoro


from animal import Animal

class Omnivoro(Animal):
    def alimentarse(self):
        print(f"{self.nombre} se alimenta de todo.")

Uso de las clases


from carnivoro import Carnivoro
from herbivoro import Herbivoro
from omnivoro import Omnivoro

leon = Carnivoro("León")
vaca = Herbivoro("Vaca")
oso = Omnivoro("Oso")

leon.alimentarse()
vaca.alimentarse()
oso.alimentarse()

Resultado:


León se alimenta de carne.
Vaca se alimenta de plantas.
Oso se alimenta de todo.

Este es un ejemplo claro de cómo se pueden usar las clases y la herencia para crear estructuras jerárquicas y reutilizar código.
