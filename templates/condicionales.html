Condicionales en Python

Las estructuras condicionales permiten que tu programa tome decisiones. Según el valor de una variable o el resultado de una comparación, Python puede ejecutar un bloque de código u otro.

Las palabras clave que se utilizan son:

  • if – ejecuta un bloque si la condición es verdadera.
  • elif – evalúa una condición adicional si la primera no se cumple.
  • else – se ejecuta si ninguna de las condiciones anteriores es verdadera.

Ejemplo real:


edad = int(input("Introduce tu edad: "))

if edad >= 18:
    print("Eres mayor de edad.")
elif edad >= 13:
    print("Eres un adolescente.")
else:
    print("Eres un niño.")

¿Qué está haciendo este código?

Primero se pide al usuario su edad. Luego:

  • Si tiene 18 años o más, se considera adulto.
  • Si tiene entre 13 y 17, se considera adolescente.
  • En cualquier otro caso (menor de 13), se considera niño.

Resultado esperado (si el usuario introduce 15):


Introduce tu edad: 15
Eres un adolescente.

Las condicionales son esenciales para controlar el flujo de tu programa. Aprender a combinarlas bien te permitirá crear lógica más compleja y dinámica.
