Bienvenido a la Guía de Python

Esta guía ha sido creada con Flask y Bootstrap para ayudarte a repasar y consolidar todo lo aprendido en clase sobre Python.

A lo largo de las diferentes secciones verás ejemplos reales que has trabajado, organizados en bloques temáticos:

  • Condicionales: Uso de if, elif y else.
  • Bucles: Repetición de acciones con for y while.
  • Estructuras: Listas, tuplas, diccionarios, conjuntos.
  • Comprensiones: Sintaxis avanzada para crear estructuras de datos.
  • POO - Animales: Programación orientada a objetos con ejemplos de clases y herencia.
  • POO - Vehículos: Proyecto aplicado con herencia y especialización.

Usa el menú de navegación superior para explorar cada sección.
