¿Qué es Flask?

Flask es un microframework de Python usado para desarrollar aplicaciones web de forma sencilla y rápida. Es muy flexible, ideal para aprender o construir proyectos pequeños y medianos.

Instalación


pip install flask

Estructura básica de una app Flask


from flask import Flask

app = Flask(__name__)

@app.route('/')
def inicio():
    return "Hola, Flask"

if __name__ == '__main__':
    app.run(debug=True)

Uso de plantillas (templates)

Una de las grandes ventajas de Flask es poder separar el código Python de la parte visual usando plantillas HTML con Jinja2:


from flask import Flask, render_template

@app.route('/inicio')
def inicio():
    return render_template('inicio.html')

Ejemplo real del proyecto

Fragmento de tu archivo main.py:


@app.route('/poo_vehiculos')
def poo_vehiculos():
    return render_template('poo_vehiculos.html')
