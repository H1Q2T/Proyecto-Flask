Estructuras de Datos en Python

Python ofrece estructuras muy potentes para guardar y manipular colecciones de datos. Aquí veremos las más comunes: listas, tuplas, diccionarios y conjuntos.

Listas

Son colecciones ordenadas y modificables. Puedes añadir, eliminar o acceder a elementos fácilmente.


nombres = ["Ana", "Luis", "Carlos", "Marta"]
print(nombres[0])  # Accede al primer elemento
nombres.append("Lucía")  # Añade un nuevo elemento
print(nombres)

Resultado:


Ana
['Ana', 'Luis', 'Carlos', 'Marta', 'Lucía']

Tuplas

Similares a las listas, pero inmutables: no se pueden modificar después de crearse.


coordenadas = (10, 20)
print("X:", coordenadas[0])
print("Y:", coordenadas[1])

Resultado:


X: 10
Y: 20

Diccionarios

Guardan pares clave: valor. Muy útiles para representar información estructurada.


alumno = {
    "nombre": "Mario",
    "edad": 18,
    "curso": "DAM2"
}
print(alumno["nombre"])
alumno["edad"] = 19
print(alumno)

Resultado:


Mario
{'nombre': 'Mario', 'edad': 19, 'curso': 'DAM2'}

Conjuntos (sets)

Son colecciones desordenadas que no permiten elementos duplicados. Se usan para operaciones de conjunto.


colores = {"rojo", "verde", "azul", "rojo"}
print(colores)
colores.add("amarillo")
print(colores)

Resultado:


{'azul', 'verde', 'rojo'}
{'amarillo', 'azul', 'verde', 'rojo'}
