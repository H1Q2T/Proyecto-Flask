Estructuras de Datos en Python

Python ofrece varias estructuras de datos integradas que permiten almacenar y manipular colecciones de elementos de forma eficiente.

Listas

Las listas son estructuras ordenadas y mutables que permiten almacenar múltiples elementos.


nombres = ["Ana", "Luis", "Carlos", "Marta"]
print(nombres[0])  # Accede al primer elemento
nombres.append("Lucía")  # Añade un nuevo elemento
print(nombres)
    

Ana
['Ana', 'Luis', 'Carlos', 'Marta', 'Lucía']

Tuplas

Las tuplas son como las listas, pero inmutables. No se pueden modificar después de su creación.


coordenadas = (10, 20)
print("X:", coordenadas[0])
print("Y:", coordenadas[1])
    

X: 10
Y: 20

Diccionarios

Los diccionarios almacenan datos en pares clave-valor. Son muy útiles para representar objetos o registros.


alumno = {
    "nombre": "Mario",
    "edad": 18,
    "curso": "DAM2"
}
print(alumno["nombre"])
alumno["edad"] = 19
print(alumno)
    

Mario
{'nombre': 'Mario', 'edad': 19, 'curso': 'DAM2'}

Conjuntos (sets)

Los sets son colecciones no ordenadas y sin elementos duplicados. Se usan principalmente para operaciones de conjunto.


colores = {"rojo", "verde", "azul", "rojo"}
print(colores)
colores.add("amarillo")
print(colores)
    

{'azul', 'verde', 'rojo'}
{'amarillo', 'azul', 'verde', 'rojo'}
