Comprensiones en Python

Las comprensiones permiten crear estructuras de datos como listas, diccionarios o conjuntos de forma compacta y eficiente.

List Comprehension

Ejemplo real para generar una lista de cuadrados:


cuadrados = [x**2 for x in range(1, 6)]
print(cuadrados)
    

[1, 4, 9, 16, 25]

Con filtrado:


pares = [x for x in range(10) if x % 2 == 0]
print(pares)
    

[0, 2, 4, 6, 8]

Dict Comprehension

Ejemplo real para crear un diccionario con pares número-cuadrado:


cuadrados_dict = {x: x**2 for x in range(1, 6)}
print(cuadrados_dict)
    

{1: 1, 2: 4, 3: 9, 4: 16, 5: 25}

Set Comprehension

Ejemplo para crear un conjunto sin duplicados:


letras = {letra for letra in "banana"}
print(letras)
    

{'b', 'a', 'n'}

Las comprensiones son una forma poderosa y concisa de trabajar con colecciones en Python.
