Bienvenido a la Guía de Python

Este sitio ha sido desarrollado con Flask y Bootstrap, con el objetivo de ayudarte a repasar y entender los fundamentos de Python a través de ejemplos reales que ya has trabajado en clase.

A continuación encontrarás una serie de secciones que explican cada concepto paso a paso, acompañadas de código comentado y resultados esperados:

  • Condicionales: Cómo tomar decisiones en tu código con if, elif y else.
  • Bucles: Repetición de acciones usando for y while.
  • Estructuras de datos: Manejo de listas, tuplas, diccionarios y sets.
  • Comprensiones: Crear listas, conjuntos y diccionarios de forma compacta.
  • POO - Animales: Introducción a la programación orientada a objetos con clases e herencia.
  • POO - Vehículos: Proyecto práctico de herencia y composición con clases reales.
  • Flask: Explicación del framework web usado para desarrollar esta guía.

Usa el menú superior para navegar entre secciones. ¡Explora, estudia y aprende a tu ritmo!
